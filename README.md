## ダウンロード

https://github.com/KanashisaBlue/OneVP-Installer/releases  
↑こちらのページから最新の .dmgファイル をダウンロードしてインストールしてください。

このソフトはMacOS専用です。Windowsでは動作しません。  

## 動作環境
macOS 14 Sonoma以降  
（Apple silicon、Intel両対応ユニバーサルバイナリ）

## このアプリケーション上での設定

VOICEPEAKをデフォルト以外の場所にインストールしている場合は、  
起動直後にダイアログが立ち上がるので、VOICEPEAKのアプリケーションを指定してください。  

## わんコメ側の設定

### 読み上げの停止

わんコメ側の読み上げを止める必要があります。  
枠オプション（枠欄の一番右の「…」→「設定」）の「読み上げボリューム」のスライダーを一番左まで寄せて読み上げを切ってください。

<img width="724" height="459" alt="image" src="https://github.com/user-attachments/assets/b256a0c9-77a9-440f-b978-b245607d387f" />

### その他の設定
  
「設定」→「読み上げ」タブの「読み上げボリューム」「読み上げ速度」「読み上げピッチ」は使用できません。  
それ以外の設定はおおむね使用できます。

## おまけ機能

話者に「商用可能 6ナレーターセット」「宮舞モカ」「フリモメン」を指定している場合、  
「絵文字で感情/性別を変更」にチェックを入れると、コメント内に特定の絵文字がある場合に  
発生ボイスの感情や性別が変わる機能があります。  

|絵文字|内容|
|---|---|
|😊😢😆😡😶|感情表現が入る|
|👨👩|強制的に絵文字の性別に変わる|

## 注意点
Lttle Snitchなどのアプリケーションでネットワーク通信を監視するソフトを入れている場合、  
このアプリケーション（OneVP）の外部へのネットワークを遮断してしまうと  
VOICEPEAKでエラーが出て読み上げができない場合があります。  
（VOICEPEAK側に外部へ通信する処理が入っているためです。このアプリケーション自体は外部への通信処理はありません）  

Blackholeなどの仮想オーディオデバイスをお使いの場合は、  
MacOS標準機能の「Audio MIDI設定」アプリで「マルチ出力デバイス」を作成し、  
システムサウンド出力をマルチ出力デバイスに設定するなどの対応が必要になる場合があります。
